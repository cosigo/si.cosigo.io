El token Cosigo

Representación digital de una ancla física explícita

El token Cosigo es una unidad digital diseñada para representar una cantidad específica de plata registrada dentro del sistema Cosigo.

Cada token Cosigo corresponde a una relación explícita, documentada y verificable con una unidad física.

Relación con la ancla

El sistema Cosigo se basa en una correspondencia directa entre el token y una cantidad definida de metal físico.

1 miligramo de plata registrada = 1 Cosigo

Esta relación define una referencia, no una promesa de valor futuro ni una garantía de estabilidad económica.

Qué representa el token

  • Una unidad de referencia explícita y definida
  • Una representación digital documentada
  • Una correspondencia verificable con metal registrado

Qué no representa el token

  • No es una moneda fiduciaria
  • No es un instrumento de inversión
  • No garantiza precios ni rendimientos
  • No sustituye la custodia personal ni la diligencia debida
